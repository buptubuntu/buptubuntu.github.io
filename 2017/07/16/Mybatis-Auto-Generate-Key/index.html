关键类

  • MapperProxy

    Mybatis 所有Mapper接口的代理类

  • MapperMethod

    Mapper接口方法动态代理执行的分装类

  • ParamNameResolver

    Mybatis中在中MethodSignature中用于根据方法来确定参数名

  • SqlSessionTemplate

    mybatis的SqlSession在spring中的实现类

  • DefaultSqlSession

    mybatis中sqlsession接口的默认实现类

  • CachingExecutor

    mybatis中具体sql的执行类

  • BaseExecutor

    mybatis的Executor接口的基础实现类

  • ReuseExecutor

  • RoutingStatementHandler

    mybatis中各种StatementHandler的路由类

  • PreparedStatementHandler

    mybatis中prepare类型的statement处理实现类

  • SelectKeyGenerator

    自增主键生成之后的处理类

  • Jdbc3KeyGenerator

    默认自增主键生成类

  • MetaObject

    元数据对象

  • BeanWrapper

    普通对象包装类
